Мясопродукт

Мясопродукт — российский производитель колбас и деликатесов из мяса. Компания работает с 2008 года и в настоящий момент представляет собой известный бренд, имеющий собственное производство и сеть дистрибуции по всей территории России. Заказчик поставил задачу разработать корпоративный сайт с онлайн-каталогом, а так же реализовать ряд требований для последующей SEO-оптимизации.

Адрес проекта
myaso-product.ru
В рамках проекта выполнены следующие работы:
Прототипирование
Дизайн для 2 разрешений (десктоп + смартфон)
Верстка и интеграция с системой управления сайта
Разработка дополнительных элементов фирменного стиля
Коменда проекта:
Евгения Струк — Руководитель проекта
Евгения Струк — Дизайнер
Евгения Струк — Web-разработчик
Анализ целевой аудитории и прототипирование
Перед стартом проекта была проанализирована целевая аудитория, прогнозируемые источники трафика и желаемые действия посетителей сайта. После формализации качественных требований был разработан прототип всех основных страниц.
Дизайн-концепция
На момент старта разработки заказчик не располагал утвержденным фирменным стилем, в распоряжении дизайнера были только отдельные его носители и пожелания по цветам и стилистике. Для целей проекта были доработаны отдельные элементы фирменного стиля, доработан логотип, утверждены фирменные шрифты и цветовая палитра бренда.
Мобильная версия
Для всех страниц была разработана адаптивная версия дизайна, позволяющая удобно пользоваться сайтом с мобильных устройств.
Разработка UI-Kit
Для дальнейшего развития проекта был разработан интерфейсный файл, включающий используемые шрифты, цветовую палитру и коллекцию всех элементов дизайна.
Особенности проекта
Особенностью проекта было пожелание заказчика сделать расширенную информационную панель для индивидуальной карточки товара. Такое решение было примененое не только в связи с большим количеством разнородной информации, которую было необходимо наглядно представить на одной странице, но и с целью обеспечить прямую обратную связь между посетителями сайта и производителем с помощью системы сообщений «Вопрос-ответ».
